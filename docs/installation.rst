============
Installation
============


Stable release
--------------

To install note-editor, run this command in your terminal:

.. code-block:: console

    $ pip install note_editor

This is the preferred method to install note-editor, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for note-editor can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/chloe-dh/note_editor

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/chloe-dh/note_editor/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/chloe-dh/note_editor
.. _tarball: https://github.com/chloe-dh/note_editor/tarball/master
